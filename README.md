# MBA_research_event

Flask/HTML event-management prototype with team dashboards, login flow, SQLite storage, and admin views.

## What It Does

- Provides basic login and admin/team dashboard pages.
- Uses a local database for user or team data.
- Experiments with event/research coordination workflows.

## Tech

Python, Flask, HTML, SQLite.

## Repository Map

- `app.py`, `init_db.py` - backend and database setup.
- `admin_dashboard.html`, `team_dashboard.html`, `team_data.html` - dashboard views.
- `login.html` - login surface.

## Status

Prototype event-management app.
